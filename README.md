**JumpUpSuperstar** is a brutal 2D Java platformer. You get **3 lives** to survive elite enemies with **unique skills**. You need **extreme skills** to finish this. Die? **Skill issue!** 

Our development team expanded a core frame-rate framework tutorial into a highly customized, fast-paced arcade experience featuring advanced vector-based player physics, unpredictable enemy combat tracking routines, and demanding stage level layout configurations.

---

### 🕹️ Controls
* **A** / **D**: Move horizontally
* **Space**: Jump over gaps and enemies

### 📥 Installation & Running the Game
1. Download the compiled release binary [JumpUpSuperstar.jar](JumpUpSuperstar.jar) into your local folder.
2. Ensure you have installed the [Java 25 Runtime Engine Environment](https://adoptium.net).
3. Launch your standard device terminal or command prompt utility.
4. Execute the application manually by running the target execution command:
```bash
java -jar JumpUpSuperstar.jar
```
